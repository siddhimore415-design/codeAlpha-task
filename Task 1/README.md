E-commerse-product-analysis-dashboard- E-Commerce Product Analysis Dashboard Project Description This project is an interactive E-Commerce Product Analysis Dashboard created to analyze sales performance, customer behavior, product ratings, and profit trends across different states and categories. The dashboard helps businesses gain valuable insights for better decision-making and business growth.

The project includes data visualization techniques to represent profit distribution, customer purchases, category performance, ratings analysis, and state-wise sales insights in a clear and user-friendly format.

Key Features Sales and profit analysis Customer purchase insights Product category performance tracking State-wise profit visualization Customer rating analysis Interactive dashboard filters Data-driven business insights Tools & Technologies Used Power BI Excel / CSV Dataset Data Cleaning & Transformation Data Visualization Insights Generated Punjab has the highest overall profit Electronics category generates maximum sales and profit Most products are rated between 3 to 5 stars Top customers contribute significantly to revenue Kerala, Tamil Nadu, and Maharashtra show strong sales performance Project Objective The main objective of this project is to transform raw e-commerce data into meaningful visual insights that help understand customer trends, product performance, and business profitability.
